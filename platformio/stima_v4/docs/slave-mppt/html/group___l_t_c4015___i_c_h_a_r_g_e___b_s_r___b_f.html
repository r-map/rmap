Detailed Description

ICHARGE_BSR_BF Bit Field.

This 16-bit two's complement word is the value of IBAT (0x3D) used in calculating BSR.

  • Register: ICHARGE_BSR
  • CommandCode: 72
  • Size: 16
  • Offset: 0
  • MSB: 15
  • MASK: 0xFFFF
  • Access: R
  • Default: n/a

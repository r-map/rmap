Detailed Description

LTC4015 library file defining data conversion macros and constants used by LTC4015.c.

This file contains macros and constants which can be used to represent real-world values in the source code, with automatic compile-time conversion to LTC4015 internal register scaling.

Passing runtime variables to these macros should be avoided as it will likely result in runtime calculations accompanied by associated processor loading and memory usage.

Real time system status indicator bits

  • CommandCode: 57
  • Contains Bit Fields:
    • CHARGER_ENABLED_BF : Indicates that the battery charger is active
    • MPPT_EN_PIN_BF : Indicates the MPPT pin is set to enable Maximum Power Point Tracking
    • EQUALIZE_REQ_BF : Indicates a rising edge has been detected at the EQ pin, and an equalize charge is queued
    • DRVCC_GOOD_BF : Indicates DRVCC voltage is above switching regulator undervoltage lockout level (4.3V typ)
    • CELL_COUNT_ERROR_BF : Indicates an invalid combination of CELLS pin settings
    • OK_TO_CHARGE_BF : Indicates all system conditions are met to allow battery charger operation.
    • NO_RT_BF : Indicates no resistor has been detected at the RT pin
    • THERMAL_SHUTDOWN_BF : Indicates die temperature is greater than thermal shutdown level (160C typical)
    • VIN_OVLO_BF : Indicates VIN voltage is greater than overvoltage lockout level (38.6V typical)
    • VIN_GT_VBAT_BF : Indicates VIN voltage is sufficiently greater than BATSENSE for switching regulator operation (200mV typical)
    • INTVCC_GT_4P3V_BF : Indicates INTVCC voltage is above switching regulator undervoltage lockout level (4.3V typ)
    • INTVCC_GT_2P8V_BF : Indicates INTVCC voltage is greater than measurement system lockout level (2.8V typical)

Enable limit monitoring and alert notification via SMBALERT

  • CommandCode: 13
  • Contains Bit Fields:
    • EN_MEAS_SYS_VALID_ALERT_BF : Enable meas_sys_valid_alert
    • EN_QCOUNT_LOW_ALERT_BF : Enable coulomb counter value low alert
    • EN_QCOUNT_HIGH_ALERT_BF : Enable coulomb counter value high alert
    • EN_VBAT_LO_ALERT_BF : Enable battery undervoltage alert
    • EN_VBAT_HI_ALERT_BF : Enable battery overvoltage alert
    • EN_VIN_LO_ALERT_BF : Enable input undervoltage alert
    • EN_VIN_HI_ALERT_BF : Enable input overvoltage alert
    • EN_VSYS_LO_ALERT_BF : Enable output undervoltage alert
    • EN_VSYS_HI_ALERT_BF : Enable output overvoltage alert
    • EN_IIN_HI_ALERT_BF : Enable input overcurrent alert
    • EN_IBAT_LO_ALERT_BF : Enable battery current low alert
    • EN_DIE_TEMP_HI_ALERT_BF : Enable die temperature high alert
    • EN_BSR_HI_ALERT_BF : Enable battery series resistance high alert
    • EN_NTC_RATIO_HI_ALERT_BF : Enable thermistor ratio high (cold battery) alert
    • EN_NTC_RATIO_LO_ALERT_BF : Enable thermistor ratio low (hot battery) alert

Configuration Settings

  • CommandCode: 20
  • Contains Bit Fields:
    • SUSPEND_CHARGER_BF : Suspend battery charger operation
    • RUN_BSR_BF : Perform a battery series resistance measurement
    • FORCE_MEAS_SYS_ON_BF : Force measurement system to operate
    • MPPT_EN_I2C_BF : Enable Maximum Power Point Tracking
    • EN_QCOUNT_BF : Enable coulomb counter

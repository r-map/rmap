CHARGER_CONFIG_BITS Register.

15:3 2 1 0
n/a EN_C_OVER_X_TERM_BF EN_LEAD_ACID_TEMP_COMP_BF EN_JEITA_BF

Battery charger configuration setting

  • CommandCode: 41
  • Contains Bit Fields:
    • EN_C_OVER_X_TERM_BF : Enable C/x termination
    • EN_LEAD_ACID_TEMP_COMP_BF : Enable lead acid charge voltage temperature compensation
    • EN_JEITA_BF : Enable JEITA temperature profile
